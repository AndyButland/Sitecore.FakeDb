* Lucene.Net.dll
* Sitecore.Abstractions.dll
* Sitecore.Analytics.dll
* Sitecore.Buckets.dll
* Sitecore.ContentSearch.dll
* Sitecore.ContentSearch.Linq.dll
* Sitecore.Kernel.dll
* Sitecore.Logging.dll
* sitecore.nexus.dll
